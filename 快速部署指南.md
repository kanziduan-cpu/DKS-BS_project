# ⚡ API 服务快速部署指南

## 🚀 三步快速部署

### 方式一：自动部署（推荐）

#### Windows 用户

1. **双击运行** `deploy-and-start.bat`
2. **等待上传完成**
3. **按提示登录服务器** 启动服务

```bash
ssh root@120.55.113.226
cd /opt/warehouse-monitor/cloud-server
pm2 start server.js --name warehouse-api
```

#### PowerShell 用户

1. **右键点击** `upload-to-server.ps1`
2. **选择"使用 PowerShell 运行"**
3. **等待部署完成**
4. **按提示登录服务器** 启动服务

---

### 方式二：手动部署

```bash
# 1. 上传项目
scp -r cloud-server root@120.55.113.226:/opt/warehouse-monitor/

# 2. 登录服务器
ssh root@120.55.113.226

# 3. 进入目录
cd /opt/warehouse-monitor/cloud-server

# 4. 运行部署脚本
bash deploy-api.sh
```

---

## ✅ 验证部署

### 1. 检查服务状态

```bash
pm2 status
```

### 2. 查看日志

```bash
pm2 logs warehouse-api
```

### 3. 测试 API

```bash
curl http://120.55.113.226:3000/api/
```

## 📞 常用命令

```bash

# 查看日志
pm2 logs warehouse-api

netstat -tlnp | grep 3000
```
