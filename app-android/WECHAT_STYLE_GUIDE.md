# 微信风格界面设计说明

本文档详细说明了本应用如何实现微信风格的界面设计。

## 设计原则

### 1. 配色方案
- **主色调**: 微信绿色 (#07C160)
- **背景色**: 浅灰背景 (#F5F5F5)
- **卡片背景**: 纯白 (#FFFFFF)
- **边框色**: 浅灰分割线 (#E5E5E5)
- **文字色**: 
  - 主要文字 (#000000)
  - 次要文字 (#999999)
  - 提示文字 (#CCCCCC)
- **警告色**: 红色 (#FA5151)

### 2. 字体规范
- **标题**: 18sp, 粗体
- **主要文字**: 16sp, 常规
- **次要文字**: 14sp, 常规
- **提示文字**: 12sp, 常规

### 3. 间距规范
- **卡片间距**: 12dp
- **卡片内边距**: 16dp
- **列表项间距**: 8dp
- **元素间距**: 4dp-12dp

### 4. 布局特点
- **零阴影**: 使用扁平化设计，无或极小阴影
- **1dp边框**: 使用细边框分隔元素
- **圆角**: 卡片圆角8dp，按钮圆角4dp
- **图标**: 使用矢量图标，支持主题色着色

## 组件样式

### 卡片组件 (MaterialCardView)
```xml
<com.google.android.material.card.MaterialCardView
    app:cardElevation="0dp"           ← 零阴影
    app:strokeColor="@color/card_border" ← 边框色
    app:strokeWidth="1dp"              ← 1dp边框
    android:layout_margin="12dp">        ← 12dp间距
```

### 列表项
- **高度**: 适合单手操作
- **内边距**: 16dp
- **图标**: 48dp圆形或方形
- **间距**: 元素之间4-12dp
- **分割线**: 0.5dp灰色，左缩进52dp

### 按钮
- **主要按钮**: 绿色填充，圆角4dp
- **次要按钮**: 绿色描边，圆角4dp
- **危险按钮**: 红色描边，圆角4dp
- **小按钮**: 文字按钮或图标按钮
- **最小触控区域**: 48dp

### 徽章 (Badge)
```xml
<TextView
    android:background="@drawable/status_badge_background"
    android:padding="2dp 8dp"         ← 紧凑内边距
    android:textSize="12sp"              ← 小字体
    android:textColor="@color/white"    ← 白色文字
    android:backgroundTint="@color/primary" /> ← 主题色
```

## 交互特点

### 1. 触控反馈
- 所有可点击元素使用 `?attr/selectableItemBackground`
- 点击时有波纹效果
- 长按有视觉反馈

### 2. 滚动行为
- 使用NestedScrollView实现平滑滚动
- SwipeRefreshLayout支持下拉刷新
- 横向滚动使用RecyclerView

### 3. 状态指示
- 在线状态: 绿色徽章
- 离线状态: 灰色徽章
- 未处理报警: 红色图标+灰色背景
- 已处理报警: 普通显示

### 4. 加载状态
- 下拉刷新动画
- 数据加载中显示进度条
- 空状态显示提示文字和图标

## 图标规范

### 图标尺寸
- **列表图标**: 24dp
- **头像图标**: 48-60dp
- **小图标**: 16-20dp

### 图标着色

## 实现要点

### 1. 避免使用阴影
微信设计风格强调扁平化，避免使用或最小化阴影效果。

### 2. 使用细边框
使用1dp的细边框替代阴影来区分卡片。

### 3. 统一间距
保持12dp的卡片间距，16dp的内边距。

### 4. 简洁图标
使用简单的矢量图标，支持主题色着色。

### 5. 清晰层次
通过字体大小、颜色深浅、间距来建立视觉层次。

### 6. 大触控区域
确保所有可点击元素至少48dp的触控区域。
