## 🔄 完整数据流

```
┌─────────────┐     HTTP POST /api/login
│ Android App │─────────────────────────────────►
│  (移动端)    │                              ┌──────────────┐
└─────────────┘◄──────────────────────────────│ 后端API服务  │
                                               │  8001端口    │
         ◄─────────────────────────────────────│              │
         返回token                             └──────────────┘
                                                      │
                                                      │
         MQTT Subscribe sensor/data                   │
         ◄──────────────────────────────────────────────┤
                                                      │
         MQTT Publish sensor/data                      │
┌─────────────┐◄──────────────────────────────────────│
│ Android App │        MQTT消息转发                    │
│             │◄───────────────────────────────────────│
└─────────────┘                                       │
         │                                           │
         │ MQTT Publish sensor/control                │
         │                                           ▼
         │                                    ┌──────────────┐
         └─────────────────────────────────────│ MQTT Broker  │
                                              │  1883端口    │
         MQTT Forward sensor/data              └──────────────┘
         │                                           │
         ▼                                           │
┌─────────────┐     MQTT Publish sensor/data        │
│  单片机设备  │─────────────────────────────────────►
│  (设备端)    │
└─────────────┘
```

### 验证测试 (建议)

#### 1. API接口测试
```bash
# 测试登录
curl -X POST http://43.99.24.178:8001/api/login \
  -H "Content-Type: application/json" \
  -d '{"username":"testuser","password":"123456"}'

# 测试设备注册
curl -X POST http://43.99.24.178:8001/api/register \
  -H "Content-Type: application/json" \
  -d '{
    "username":"test_device_android",
    "device_id":"android_test_001",
    "device_type":"sensor"
  }'
```

#### 2. MQTT连接测试
```bash
# 订阅传感器数据
mosquitto_sub -h 43.99.24.178 -p 1883 \
  -u testuser -P 123456 \
  -t "sensor/data" -v

# 发布测试数据 (另开终端)
mosquitto_pub -h 43.99.24.178 -p 1883 \
  -u testuser -P 123456 \
  -t "sensor/data" \
  -m '{"temp":25,"hum":60,"deviceId":"test_001","timestamp":1678234567890}'
```
