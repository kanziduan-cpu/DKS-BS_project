## 🔍 常见问题排查

### 问题1: 无法连接到MQTT

**错误信息**:
```
E/MqttManager: MQTT连接失败: Connection lost
```

**排查步骤**:
1. 检查服务器IP是否正确: `43.99.24.178`
2. 检查端口是否开放: `telnet 43.99.24.178 1883`
3. 检查用户名密码: `testuser / 123456`
4. 检查网络连接

### 问题2: 订阅主题后收不到数据

**错误信息**:
```
D/MqttManager: 订阅主题成功: sensor/data
// 但没有收到数据
```

**排查步骤**:
1. 确认订阅主题正确: `sensor/data`
2. 使用mosquitto_sub验证是否有数据发布
3. 检查设备端是否发布到正确主题
4. 查看MQTT Broker日志

### 问题3: API接口调用失败

**错误信息**:
```
Failed to connect to /43.99.24.178:8001
```

**排查步骤**:
1. 检查API端口是否为 `8001`
2. 测试API是否可访问: `curl http://43.99.24.178:8001/api/health`
3. 检查防火墙规则
4. 查看服务器日志

### 问题4: Clean Session 配置问题

**错误现象**: 频繁断线或无法接收离线消息

**解决方案**:
- 服务器配置: `Clean Session: true`
- Android配置: `CLEAN_SESSION = true`
- 两者保持一致
